# Expense Tracker (Python)

## Description
A command-line based expense tracker built using Python.
It uses OOP concepts and file handling to store data permanently.

## Features
- Add expenses by category
- View total expenses
- Persistent storage using files
- Clean and modular code

## Concepts Used
- Loops
- Functions
- OOP
- File Handling

## Future Improvements
- Menu-driven UI
- Date-wise expense tracking
- Input validation
